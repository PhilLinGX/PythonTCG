项目名：TCG彩票自动分析投注
项目备份：github(第一次使用)

最初目标：用Python+QT搭建用户界面，登陆亚博/TCG彩票(1分钟/1.5分钟)，自动下载历史彩票情况，并将整合成一个txt(可以独立一个工程放在云端每天自动完成)。通过对彩票的历史分析(遗漏，冷热，全天平衡，概率分析等)并进行数据可视化，然后通过判断进行投注。将所有投注和盈利情况进行汇总显示。力求达到盈利(答案估计是否定的)。

初步设想框架：用pyQt先创建用户界面，然后再去学习网络爬虫(浏览器点击后具体做了些什么？怎么去达到自动投注的功能？)，最后再进行数据处理与分析。
